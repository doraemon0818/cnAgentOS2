# 需求文档

> 本文档用于记录和跟踪 `AI智能瞭望、智能问数与智能聊天系统` 的当前需求、完成状态与后续规划。
> 本文档与 `README.md` 配套维护，以“当前代码实际状态”为准。
> 状态说明：`✅ 已完成`、`🔄 部分完成/持续迭代`、`⬜ 未开始`

---

## 技术栈

| 类别 | 技术/组件 | 版本 | 说明 |
|------|-----------|------|------|
| 后端框架 | Tornado | 6.5.5 | 异步非阻塞 Web 框架 |
| 编程语言 | Python | 3.11+ | 主开发语言 |
| 数据库 | SQLite3 | 内置 | 当前主存储，预留扩展 |
| 前端 UI 框架 | Layui | 2.13.6 | 后台管理界面 |
| 前端 CSS 框架 | Bootstrap | 5.3.8 | 前台页面布局补充 |
| 图标库 | FontAwesome | 5.15.4 | 本地图标资源 |
| 模板引擎 | Tornado Template | 内置 | 服务端模板渲染 |
| 模型接入 | OpenAI Compatible API | - | 兼容云端/本地模型服务 |
| 采集解析 | requests / bs4 / lxml / raw4ai | - | 瞭望采集与深度采集 |
| 虚拟环境 | venv | Python 3.11 | 独立运行环境 |
